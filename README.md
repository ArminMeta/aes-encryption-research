**AES Algorithm – Encryption of Text Documents**

This repository contains my graduation thesis titled "Application of the AES Algorithm for Encrypting Text Documents", along with example concepts and explanations of how the AES cryptographic algorithm works. The thesis explores the fundamentals of cryptography, the structure of the Advanced Encryption Standard (AES) algorithm, and its practical application in encrypting textual data using programming tools.

Author: Armin Bojanić Graduation Thesis – Programming

**Overview**

With the rapid growth of digital technologies, protecting sensitive data has become increasingly important. Information that was once stored physically is now kept in digital form, which requires reliable security mechanisms. One of the most widely used encryption algorithms today is the Advanced Encryption Standard (AES). AES provides a high level of security and efficiency and has become a global standard for protecting digital information.

The goal of this thesis is to: Explain the importance of encryption in the digital world Describe the principles and structure of the AES algorithm Demonstrate practical encryption of text documents Topics Covered

**The thesis includes the following topics:**

Cryptography and Data Security
